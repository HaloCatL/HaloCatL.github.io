奇数码游戏是它的一个扩展，在一个$n×n$的网格中进行，其中$n$为奇数，1个空格和$1$到$n^2−1$这$n^2−1$个数恰好不重不漏地分布在$n×n$的网格中。

空格移动的规则与八数码游戏相同，实际上，八数码就是一个$n=3$的奇数码游戏。

现在给定两个奇数码游戏的局面，请判断是否存在一种移动空格的方式，使得其中一个局面可以变化到另一个局面。

再接下来$n$行每行$n$个整数，表示第二个局面。

局面中每个整数都是$0$到$n^2−1$之一，其中用$0$代表空格，其余数值与奇数码游戏中的意义相同，保证这些整数的分布不重不漏。

输出格式

对于每组数据，若两个局面可达，输出TAK，否则输出NIE。

奇偶性很神奇，对于一类问题，如果属于同种性质（奇偶性相同），那么它们就是完全相同（这个在某种意义上说）的，一些特殊的情况又例外，对于这道题，我们用题目给的例子来讲解。 为了好看，这里我使用 # 来表示空格方块 （左边图我们编号为①）

到此基础的看题入手的思路就完成了（一些移动问题都可以弄成一维来看，环的话可以断成链来考虑），接下来就是使用一些算法来解决我们的当前需要解决的问题，因为我们这里根据移动三个数字，移动一个数字来考虑的话，就是需要一个计数类的算法来解决，目前所学到的针对一维数列的有…只有逆序对（后面学到新的我会回来补为什么不选其他的）

那么我们就针对逆序对来思考这道题，首先我们先将逆序对的数量算出来（不考虑空格位置）

①：2 ②：4 ③：4 ④：4 ⑤：4

先针对①②③，我们发现 左右移动完全不改变逆序对个数，而上下移动并不改变逆序对的奇偶性（这里为了节省时间不列举更多的例子，需要更深入的理解请自行再多次进行移动操作）

再针对①④⑤，我们发现也是如此

那么我们只需要使初始图形的逆序对个数和结束图形的逆序对个数相同，就一定有一种方案可以将初始图形改变到结束图形，反之则不可以。（到此，结束） （这里如果不理解为什么逆序对（奇偶性）相同就一定能拼到最终图形，有两种方案可以解除你的疑问 ①：自欺欺人记结论法还记得我之前说，一些问题如果奇偶性相同那么里面涉及的问题都是等价的吗，这里也可以这么做 ②：小小的列举法，我们这里每个数都可以进行移动，因为每次移动一次或者三次，那么一定可以遍历完所有的数与可以让所有的数移动，或者你可以等价成数字之间的移动，这样的话每个数字都一定能够移动，每次移动奇偶性一定是不变的，所以可以证得，如果奇偶性相同一定能从初状态转移到末状态，反之则不可以）
