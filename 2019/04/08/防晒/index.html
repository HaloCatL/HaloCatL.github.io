防晒

发表于 2019-04-08 | 分类于 OI
字数统计: 1.2k | 阅读时长 ≈ 4

有C头奶牛进行日光浴，第i头奶牛需要minSPF[i]到maxSPF[i]单位强度之间的阳光。

每头奶牛在日光浴前必须涂防晒霜，防晒霜有L种，涂上第i种之后，身体接收到的阳光强度就会稳定为SPF[i]，第i种防晒霜有cover[i]瓶。

求最多可以满足多少头奶牛进行日光浴。

输入格式

第一行输入整数C和L。

接下来的C行，按次序每行输入一头牛的minSPF和maxSPF值,即第i行输入minSPF[i]和maxSPF[i]。

再接下来的L行，按次序每行输入一种防晒霜的SPF和cover值,即第i行输入SPF[i]和cover[i]。

每行的数据之间用空格隔开。

输出格式

输出一个整数，代表最多可以满足奶牛日光浴的奶牛数目。

数据范围

$1≤C,L≤2500,$
$1≤minSPF≤maxSPF≤1000,$
$1≤SPF≤1000$

题目题解：

做法：

  • 将牛按最小SPF从大到小排序，将防晒霜从大到小排序

  • 然后将防晒霜依次分配给牛，大的优先分配给大的

证明：

此题证明需要用到增广路来作为理论依据，增广路中我们需要用到最短增广路

首先，我们先将此题的模型转化，此题可以转化成线与点的有关模型，牛可以转化为线，给的SPF可以转化为点，那么我们借于点和线的关系，我们就可以将其转化为一个二分图最大匹配的有关问题（不了解二分图最大匹配的可以先去学习一下匈牙利算法，此算法专门用于解决二分图最大匹配，此篇因为篇幅有限，不再阐述此算法，后面学习到的时候会专门写出来），那么转化为二分图的问题之后，我们就可以用我们的增广路性质了

增广路的定义（详细请了解有关增广路的五条结论）

假设我们此时存在增广路，那么我们来看最短的一条增广路，为了方便，我们用 $C_i$ 表示第 $i$ 头牛，用 $S_i$ 表示第 $i$ 个防晒霜。

假设我们此时选择的最短增广路是 $S_{i_0} -> C_{i_1}->S_{i_2} -> C_{i_3}->…->C_k $ 此时$S_{i_0}$与$C_{i_k}$均是不在匹配中的点。

我们考虑增广路径的起点 $S_{i_0}$ 可以发现：

  • $S_{i_0} ≤ S_{i_2}$，这是因为我们每次会给每头奶牛匹配一个符合条件的尽可能大的防晒霜（为什么？ 增广路性质）（从这里我们能得到牛一定是从大到小，防晒霜一定是从大到小 以上两种依次匹配）
  • $i_1 > i_3$，如果 $i_1 < i_3$，我们就能将 $c_{i_1}$ 和 $s_{i_2}$ 从增广路中删除（因为不满足最短增广路定义），此时就和我们定义的最短增广路所冲突 （从这里我们能得到该序列一定是单调的（排序可得））

根据上面的推论，同理，我们推导后面一样可以得到

  • $S_{i_j} ≤ S_{i_j + 2}$
  • $i_j > i_{j + 2}$

所以可得 $i_{k - 2} > i_k$，所以在上述做法中，我们一定会先枚举到$C_{i_k}$再枚举到$C_{i_{k - 2}}$，与 $C_{i_k}$没有匹配到任何防晒霜相矛盾（因为我们先枚举大的后枚举小的），所以上述做法一定不存在增广路径，所以上述做法得到的匹配一定是最大匹配

由此我们可以得到，一定是按牛的$MinSPF$从大到小排序，然后与防晒霜依次匹配（防晒霜也是从大到小，因为此时符合我们的最短增广路）
