国王游戏

发表于 2019-04-10 | 分类于 OI
字数统计: 1.6k | 阅读时长 ≈ 6

恰逢 H 国国庆,国王邀请 n 位大臣来玩一个有奖游戏。

首先,他让每个大臣在左、右手上面分别写下一个整数,国王自己也在左、右手上各写一个整数。

然后,让这 n 位大臣排成一排,国王站在队伍的最前面。

排好队后,所有的大臣都会获得国王奖赏的若干金币,每位大臣获得的金币数分别是:

排在该大臣前面的所有人的左手上的数的乘积除以他自己右手上的数,然后向下取整得到的结果。

国王不希望某一个大臣获得特别多的奖赏,所以他想请你帮他重新安排一下队伍的顺序,使得获得奖赏最多的大臣,所获奖赏尽可能的少。

注意,国王的位置始终在队伍的最前面。

输入格式

第一行包含一个整数 n,表示大臣的人数。

第二行包含两个整数 a 和 b,之间用一个空格隔开,分别表示国王左手和右手上的整数。

接下来 n 行,每行包含两个整数 a 和 b,之间用一个空格隔开,分别表示每个大臣左手和右手上的整数。

输出格式

输出只有一行,包含一个整数,表示重新排列后的队伍中获奖赏最多的大臣所获得的金币数。

数据范围

$1≤n≤1000$
$0<a,b<10000$

先说做法：
  • 先将每个大臣按左手乘右手的顺序从小到大排序
  • 然后用题面所给的方法计算，每次取最大值即可，那便是最优结果
证明：

假设我们有一段未处理过的数列 ${C_n}$ ,并设这个数列中每一个位置都有两个值，分别对应左手右手 即 $A_i B_i$ 与 $A_{i + 1}B_{i+1}$ ，那么对于这个数列 ${C_n}$ 一定会有一个连续的两个数满足 $A_iB_i$ > $A_{i + 1} B_{i + 1}$

现在将两个数的位置交换一下，我们看看最后能得到的结果相比于之前是怎么样的。

交换前： 第$i$个人是 $\frac{\prod ^{i - 1}_{j = 0} A_j}{B_i}$ 第$i + 1$个人是 $\frac{A_i * \prod ^{i - 1}_{j = 0} A_j}{B_{i + 1}}$

交换后： 第$i$个人是 $\frac{\prod ^{i - 1}_{j = 0} A_j}{B_{i + 1}}$ 第$i + 1$个人是 $\frac{A_{i + 1} * \prod ^{i - 1}_{j = 0} A_j}{B_i}​$

我们将交换前后的式子全部都除一个$\prod ^{i - 1}_{j = 0} A_j$，然后再乘一个$B_i * B_{i + 1}$ 可得

第$i$个人 第$i + 1$个人
交换前 $B_{i + 1}$ $A_i * B_i$
交换后 $B_i$ $A_{i+1}*B_{i + 1}$

由于我们严格定义了 $A_i B_i > A_{i + 1} B_{i + 1}$ 且 $A_i B_i > B_i$ ，于是我们就可以知道交换前的 $A_i B_i≥Max(B_i, A_{i + 1} * B_{i + 1})$ 这样我们的结果一定是越来越优（越来越小） 而且这两个交换位置并不会对整个序列造成影响，所以一旦存在逆序，将其交换，得到的结果一定不会比原来差。

所以从小到大排序就可以得到最优解，证毕。
